### **DESCRIÇÃO DO PROJETO**

Check-List criada em python contendo todas as operações básicas do CRUD (CREATE, READ, UPDATE, DELETE)

### **FUNCIONALIDADES**
* Adicionar tarefas com descrição e data de vencimento
* Listar todas as tarefas
* Filtrar por status (pendente / concluída)
* Marcar tarefas como concluídas
* Remover tarefas
* Versão opcional com interface gráfica (Tkinter)

### **REQUISITOS**

* Python 3.x instalado na máquina
* Windows 10 ou 11
